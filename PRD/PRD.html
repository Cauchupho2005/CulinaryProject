Business Objectives

Mục tiêu #1 - Tăng trải nghiệm du khách

Thuyết minh tự động 5 ngôn ngữ, không cần tai nghe/guide. Giảm 80% thời gian chờ hướng dẫn viên.

Mục tiêu #2 - Hỗ trợ offline hoàn toàn

Tải 1 lần ~50MB (POI + audio + maps), dùng mãi mãi không cần mạng.

Heatmap footfall, thời gian nghe, lộ trình di chuyển → tối ưu vị trí/menu.

Mục tiêu #4 - Zero maintenance

Web CMS tự quản POI/audio. Admin chỉ duyệt thay đổi lớn.

Chapter 05

Hệ thống Geofencing

Algorithm & Implementation

POI { lat, lng, radius: 30m, priority: 1 } BackgroundService (iOS Always, Android FusedLocation) watchPosition(highAccuracy, throttle: 5s) if (Haversine(userPos, poiPos) <= radius && !inside) { debounceTimer = 3s } if (debounceComplete) triggerAudio() cooldown = 5min after exit
GPS Update (5s) → Haversine Calc → Debounce 3s → TTS Play

Chapter 13

End-to-End Flow — Từ Mở App đến Nghe Audio

Luồng hoàn chỉnh cho Du khách (Happy Path):

1. Mở App lần đầu

Chào mừng → Permissions (GPS + Storage) → Download Pack 50MB (POI + Audio + Maps). Tiến trình progress bar. Hoàn tất → Chọn ngôn ngữ mặc định (VN/EN detect).

2. Khởi động GPS Service

Background service start. Map view hiện Phố Vĩnh Khánh + 50+ POI markers. User location blue dot. Auto-fit bounds.

3. Di chuyển đến POI #1 (30m radius)

GPS update mỗi 5s → Haversine distance <30m → Enter geofence detected → Debounce timer 3s (avoid false trigger).

4. Audio Trigger (4-Tier Fallback)

Tier1: Pre-gen MP3 từ SQLite (0ms) → TTS Player popup (title + progress + pause/skip) → Auto-play sau 1s. Volume system default.

5. Di chuyển POI tiếp theo

Exit geofence → Cooldown 5min → Next POI trigger. Route line nối các POI đã visit. Heatmap update (online).

6. Kết thúc Tour

Summary screen: POI visited, total distance, time spent. Share route. Background sync analytics.

App Open → Download Pack → GPS + Map → Geofence Enter (30m) → TTS Auto Play → Next POI
Error Handling: No GPS → QR scan POI. No audio → Text popup + Native TTS. Low battery → GPS throttle 30s. Offline → Cached content only.
